Python Special Methods (Dunder Methods)

A Jupyter Notebook covering the special methods (also called magic methods or dunder methods) in Python's Object-Oriented Programming, with notes and working code examples for each.

## What's in the notebook

The notebook walks through 12 special methods:

1. `__init__()` – Constructor
2. `__str__()`, `__repr__()` – String representation
3. `__len__()` – Length
4. `__eq__()` – Equality
5. `__add__()`, `__sub__()` – Operator overloading
6. `__getitem__()` – Item access
7. `__setitem__()` – Item assignment
8. `__contains__()` – Membership testing
9. `__call__()` – Callable objects
10. `__iter__()`, `__next__()` – Iteration support
11. `__bool__()` – Boolean evaluation
12. `__del__()` – Destructor

Each one has a short explanation followed by a runnable code example demonstrating it.

## How to use

1. Clone the repo:
   ```
   git clone <repo-url>
   ```
2. Open the notebook in Jupyter:
   ```
   jupyter notebook special_methods.ipynb
   ```
3. Run the cells in order from top to bottom.

## Requirements

- Python 3.x
- Jupyter Notebook (or JupyterLab / VS Code with Jupyter extension)
